Server Requirement

Before starting the Cord-in-a-Box installation, be sure that your server fit the following requirement

  • 64-bit AMD64/x86-64 server
  • 48 GB+ RAM
  • 12+ CPU cores
  • 1 TB+ disk
  • Access to internet
  • ubuntu 16.04.3 with fresh update
  • sudo less
